Accounts


Welcome to the account management page! This region of the application automatically retrieves and displays any accounts tied to the active system user, and it provides functionality for adding, updating, and deleting those accounts. When adding an account, the Account ID and Account Name are required, and the balance will default to $100 unless otherwise specified. When updating an account, the Account ID and Account Name are required, and only the value of the Account Name can actually be updated. When deleting an account, only the Account ID is required, and the other input fields will be ignored.

Relevant Business Rules:
An account cannot have less than $100 at any time.
