Transactions


Welcome to the transaction management page! This region of the application automatically retrieves and displays any accounts tied to the active system user, and it provides functionality for processing withdrawals and deposits for a specified account. The Account ID and Transaction Amount are both required to perform either transaction.

Relevant Business Rules:
An account cannot have less than $100 at any time.
A user cannot withdraw more than 90% of their total balance from an account in a single transaction.
A user cannot deposit more than $10,000 in a single transaction.
